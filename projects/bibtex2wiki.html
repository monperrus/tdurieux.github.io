# BibTex2Wiki

BibTex2Wiki is a simple library that transforms BibTex references into Wikipedia references. This tools can be used to simplify the redaction of scientific articles in Wikipedia. The source code of the tool is available on GitHub and the tool is here: https://durieux.me/synctex-js/.
