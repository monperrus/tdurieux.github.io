Thomas Durieux Projects

BanditRepair

BanditRepair is framework that automatically handles null pointer exceptions at runtime. This framework also explores the repair space in order discover the best available repair decision.

IntroClassJava

IntroClassJava is a dataset of small buggy Java programs. This dataset is the Java version of the original dataset IntroClass that contains small buggy C programs.

\BlueLaTeX

\BlueLaTeX aims to provide a tool chain to easily write collaboratively LaTeX documents. I developed the new UI interface using the MVC framework Angular.js.

DynaMoth

DynaMoth is a dynamic code synthesizer designed for automatic repair. This synthesizer uses an oracle and the runtime context to synthesize Java expressions.

NoPol

NoPol is an automatic software repair tool developed at INRIA Lille. This tools aims to fix buggy condition and missing precondition.

Dependency Analyzer

Java Dependency Analyzer is a tool for listing all dependencies of a project at different level of abstraction (package, class and method).This tool uses Spoon for analyzing the source code of a Java project.

SyncTeX JS Parser

SyncTeX JS Parser is a SyncTeX parser written in JavaScript. This parser is used in the \BlueLaTeX project.

MultiAgent

SyncTeX JS Parser is a SyncTeX parser written in JavaScript. This parser is used in the \BlueLaTeX project.

BibTeX To Wikipedia

BibTeX2Wiki is a small tool which transforms BibTeX file into a Wikipedia formated references.
